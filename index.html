FAQ

常见问题

转换会上传我的文件吗？

不会。markify 的转换引擎完全在本地运行，文件不会上传到任何服务器，断网也能正常使用。

支持哪些格式？

Word（.docx/.doc）、Excel（.xlsx/.xls/.csv）、PowerPoint（.pptx/.ppt）、数字版 PDF、HTML、以及常见图片格式。

扫描件 / 图片型 PDF 能转吗？

目前对没有文字层的扫描件 PDF 暂不支持（不会瞎编内容）。OCR 识别在后续版本规划中。

怎么激活？

在闲鱼购买后会获得激活码，打开 app 点「激活」输入即可。一码一设备，可自助换机 3 次。

支持 Windows 吗？

当前优先支持 macOS，Windows 版在规划中。

导出的 Markdown 里有图片吗？

有。导出时图片会自动打包到与 .md 同名的目录，拖进 Obsidian、Typora 等都能正常显示。
